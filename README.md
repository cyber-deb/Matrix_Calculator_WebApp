# 🚀 Matrix Calculator Web Application

A modern, responsive **Matrix Calculator** built with **Python**, **Flask**, **HTML**, **CSS**, and **JavaScript**.

Perform essential matrix operations instantly through a clean web interface—no installation required.

## 🌐 Live Demo

👉 **https://matrix-calculator-ztcq.onrender.com/**

---

## ✨ Features

- ➕ Matrix Addition
- ➖ Matrix Subtraction
- ✖️ Matrix Multiplication
- 📐 Determinant Calculation
- 🔄 Matrix Inverse
- ⚡ Fast calculations
- 📱 Responsive UI
- 🎨 Modern glassmorphism-inspired design
- ✅ Input validation
- ⚠️ Helpful error messages
- 💻 Runs completely in your browser

## 🚀 Run Locally

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

Move into the project folder

```bash
cd YOUR_REPOSITORY
```

Install dependencies

```bash
pip install -r requirements.txt
```

Start the Flask server

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
